## ✨ Features

- **Multi-language Support**: Works with YouTube videos in any language (Hindi, Chinese, Spanish, etc.)
- **Reliable Transcription**: Uses Supadata API with residential proxies to bypass datacenter IP blocks
- **AI-Powered Writing**: LLaMA 3.3 70B generates well-structured articles with proper formatting
- **Web Enrichment**: Automatically finds and includes relevant context from the web
- **Fast & Free**: No signup required, powered by Groq's blazing-fast inference
- **Export to PDF**: Download your articles as beautifully formatted PDFs


## 🚀 How It Works

1. **Fetch Transcript**: Supadata API retrieves YouTube captions (works from any IP, including datacenter IPs)
2. **Translate & Process**: Multi-language transcripts are automatically translated to English
3. **Enrich Context**: Serper.dev searches for relevant web content, Firecrawl extracts clean text
4. **Generate Article**: LLaMA 3.3 70B writes a structured article with title, sections, and proper formatting
5. **Display & Export**: View in browser or export as PDF
